# 🧠 Computer Vision Projects

A collection of beginner-to-advanced computer vision projects implemented using Python, OpenCV, and deep learning techniques. This repository serves as a hands-on exploration of image processing, feature detection, and real-time vision systems.

---

## 🚀 Overview

This repository contains multiple computer vision tasks ranging from basic image processing to more advanced applications like object detection and tracking.

Computer vision is all about enabling machines to interpret and understand visual data, which has applications in fields like healthcare, surveillance, autonomous driving, and more ([Omdena][1]).

---

## 🛠️ Tech Stack

* Python 🐍
* OpenCV
* NumPy
* Matplotlib
* PIL (Python Imaging Library)
* Deep Learning (CNNs, LSTM, GRU where applicable)

---

## 📂 Projects Included

### 🔹 Image Processing

* Blur Detection (Laplacian-based)
* Contrast Detection
* Cartooning Images
* Image-to-Cartoon Conversion

### 🔹 Edge Detection

* Canny Edge Detection
* Sobel Operator
* Laplacian Edge Detection

### 🔹 Feature Detection

* Harris Corner Detection
* Shi-Tomasi Corner Detection

### 🔹 Contours & Shapes

* Contour Detection and Visualization
* Bounding Box Drawing

### 🔹 Video Processing

* Convert Video to Frames
* Frame Extraction Utilities

### 🔹 Object Detection & Tracking

* Ball Detection and Tracking
* Haar Cascade Face Detection
* Eye Detection

### 🔹 Advanced / Deep Learning

* CNN-based Models
* LSTM & GRU Experiments
* Monte Carlo Localization
